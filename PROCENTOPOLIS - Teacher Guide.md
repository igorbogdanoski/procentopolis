# ProcentOpolis - Teacher's Guide

**Author:** Igor Bogdanoski
**Version:** 1.0
**Subject:** Mathematics - Percentages
**Target Group:** 8th Grade Students (Middle School)

---

## What is ProcentOpolis?

ProcentOpolis is an **interactive educational game** inspired by Monopoly, designed for learning and practicing **percentage calculations** through fun property trading. The game works in a web browser - it requires no installation, just an internet connection.

Students learn percentages while:
- Buying properties (solving tasks to obtain them)
- Paying rent (calculating percentages correctly or paying double)
- Trading properties with each other
- Managing a budget and making financial decisions

---

## How to Start?

### Step 1: Teacher Creates a Room

1. Open the application in your browser.
2. Select the **TEACHER** role.
3. Enter your name.
4. Click **ENTER GAME** - the system automatically creates a room with a code (101, 102...).
5. The room code will be displayed on the screen.

### Step 2: Students Join

1. Students open the application.
2. They select the **STUDENT** role.
3. They enter their name and class.
4. They enter the room code (or select from the list of active rooms).
5. They choose an avatar (emoji).
6. They click **ENTER GAME**.

### Step 3: Starting the Game

- The teacher clicks **START GAME** when all students have joined.
- Optimal number: **4-6 players** per room.
- The teacher can create **multiple rooms** for parallel groups.

## Game Rules

### The Board

The board has **20 cells** arranged in a square:

| Cell Type | Color | Description |
|-----------|-------|-------------|
| **START** | Gray | Passing grants a bonus (15% of money) - if you solve a task. |
| **Properties** (16) | Blue, Green, Red, Yellow | Mathematical streets for purchase. |
| **CHANCE** | Orange | Random card - gain or loss. |
| **TAX** | Dark Gray | Tax inspection - 10% of money. |
| **JAIL/REST** | Gray | Miss 1 turn. |

### Property Groups (by Difficulty)

- **Blue** (Easy): Decimal Street, Fraction Park, 10% Square, Euclid Boulevard.
- **Green** (Medium): Algebra Avenue, Newton Neighborhood, Percentage Market, Pythagoras Tower.
- **Red** (Hard): Interest Bank, Gauss Heights, Statistical Zone, Financial Center.
- **Yellow** (Medium): Pi Roundabout, Limit Lane, Vector View, Matrix Market.

### Gameplay Flow

1. **Roll Dice** - The student clicks "ROLL" when it's their turn.
2. **Movement** - The avatar moves along the board.
3. **Cell Action:**
   - **Free Property:** Can buy if a math task is solved.
   - **Opponent's Property:** Must pay rent (correct answer = normal rent, wrong = DOUBLE).
   - **Own Property:** Can build (if they have a monopoly on the entire group).
   - **CHANCE:** Opens a card - gain or loss.
   - **TAX:** Solve a task to avoid the tax.
4. **End of Turn** - Automatically after 30 seconds or after an action is completed.

### Starting Funds
- Each student starts with **1000 coins (d)**.
- The game lasts **40 minutes**.

## Power Shop

Students can buy special powers that give them a strategic advantage:

| Power | Price | Effect |
|-------|-------|--------|
| ⚖️ **Lawyer** | 300d | Protects against tax inspection (one-time). |
| 🛡️ **Golden Shield** | 250d | Avoids paying rent to an opponent (one-time). |
| 🚀 **Nitro Dice** | 150d | The next roll has double value. |
| 🕵️ **Insider** | 500d | Purchases the next property for only 1 coin. |

### Streak Rewards
For every **3 correct answers in a row**, the student receives a **free power** (randomly selected).

---

## Trading Between Players

Students can trade properties with each other:

1. Click **TRADE** in the center of the board.
2. **Step 1:** Select your property to offer.
3. **Step 2:** Select a player and their property that you want.
4. **Step 3:** Add money (optional) and send the offer.
5. The other player can **accept** or **decline**.

## Teacher Control Panel

The teacher has access to a **detailed control panel** with:

### Real-time Statistics
- **Number of students** in the room.
- **Correct tasks** - total count.
- **Success rate** - average success percentage.

### Detailed Visualization
- **Live Status:** Remaining time, whose turn it is, total turns.
- **Wealth Distribution:** Graph showing each student's wealth.
- **Owned Properties:** How many properties each player has.
- **Question Difficulty:** Statistics for easy/medium/hard tasks.

### Student Table
For each student, the following is displayed:
- Name and class.
- Current wealth.
- Correct / Wrong answers.
- Last activity.
- Status (thinking, active...).

### Exporting Results
- Click **REPORT** to download a CSV file with all results.
- The file can be opened in Excel/Google Sheets.

### Multiple Room Management
- The teacher can create **multiple rooms** simultaneously.
- Each room is an independent game.
- All rooms are visible in the control panel.
- Can **start all** at once.

## Pedagogical Goals

### Mathematical Competencies
- Calculating percentages of a given value.
- Comparing percentages.
- Applying percentages in real situations (rent, tax, discount).
- Mental arithmetic and estimation.

### Transversal Skills
- **Financial Literacy** - budget management, investment, risk.
- **Strategic Thinking** - when to buy, when to build, when to trade.
- **Social Skills** - negotiation, trading, competition.
- **Decision Making Under Pressure** - time limit per turn.

### Differentiation
- Tasks have **3 difficulty levels** (easy, medium, hard).
- Board cells have varying difficulties.
- The teacher can choose a mode: **Easy**, **Normal**, or **Hard**.
- The system automatically adjusts the tasks.

## Tips for Teachers

1. **Before Class:**
   - Test the application beforehand.
   - Prepare devices (tablets/computers).
   - Check the internet connection.

2. **During Class:**
   - Explain the rules before starting (or use the built-in tutorial).
   - Monitor the control panel for student progress.
   - Intervene when a student struggles with tasks.
   - Encourage trading between students.

3. **After Class:**
   - Download the CSV report.
   - Discuss results with the class.
   - Identify topics that require repetition.

4. **Group Organization:**
   - Ideal: 4-6 students per room.
   - For a larger class: Create multiple rooms (e.g., 101, 102, 103).
   - Mix groups for better dynamics.

## Frequently Asked Questions

**Q: How many students can play at the same time?**
A: Up to 6 students per room. For a larger class, create multiple rooms.

**Q: What happens if a student disconnects?**
A: The game automatically saves progress. The student can return using the same name.

**Q: Can I pause the game?**
A: The game lasts 40 minutes. The teacher can manually end it with "REPORT".

**Q: Does it work on a phone?**
A: Yes, but a tablet or computer is recommended for a better experience.

**Q: Is registration required?**
A: No, there is no need to create an account. Just enter a name and enter.

**Q: Which mathematical topics are covered?**
A: Calculating percentages, percentage of value, comparing percentages, applied percentage tasks.
